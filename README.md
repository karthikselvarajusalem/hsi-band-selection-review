# Hyperspectral Band Selection Review

This repository contains code, preprocessing steps, and benchmarking framework for the paper:

"Review on Different Band Selection Approaches for Classification of Hyperspectral Images"

## Contents
- Data preprocessing scripts
- Band selection methods
- Evaluation framework

## Datasets
Public datasets used:
- Indian Pines
- Salinas
- Pavia University
- KSC

## Note
This repository is created to support reproducibility of the review study.
